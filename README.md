# Яндекс.Музыка Импортер 

Скрипт для автоматического импорта треков из текстового файла в раздел **"Мне нравится"** Яндекс.Музыки.

---

## Как использовать

### 1. **Получите токен**
   - Откройте [music.yandex.ru](https://music.yandex.ru)
   - Нажмите `F12` → вкладка **Network**
   - Найдите запрос к `api.music.yandex.net`
   - Скопируйте значение из заголовка **Authorization**

### 2. **Подготовьте файлы**
   - `token` — вставьте ваш токен  
     *📖 [Как получить токен (подробнее)](https://github.com/MarshalX/yandex-music-api/discussions/513)*
   - `My Spotify Library.txt` — список треков

### 3. **Формат файла с треками**

    Исполнитель - Название трека
    The Weeknd - Blinding Lights
    Dua Lipa - Don't Start Now


### 4. **Запустите скрипт**
```bash
python myscript.py
```

---

## Особенности

- 🔁 Треки обрабатываются **в обратном порядке**
- ⏭️ Ненайденные треки автоматически **пропускаются**
- 📝 Весь процесс **отображается в консоли**

---

## Внимание

**НЕ ПУБЛИКУЙТЕ `token`!**  
Храните его в безопасности и не передавайте третьим лицам.
