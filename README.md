# forYandex
# Яндекс.Музыка Импортер

Скрипт для автоматического импорта треков из текстового файла в "Мне нравится" Яндекс.Музыки.

# Как работает

1. Получите токен:
   - Откройте music.yandex.ru
   - F12 ? Вкладка Network
   - Найдите запрос к api.music.yandex.net
   - Скопируйте значение из заголовка Authorization

2. Создайте файлы:
   - token - вставьте токен ( как достать свой токен подробнее на https://github.com/MarshalX/yandex-music-api/discussions/513 )
   - My Spotify Library.txt - список треков

3. Формат файла с треками:
Исполнитель - Название трека

4. Запустите: python myscript.py

Особенности:
- Треки обрабатываются в обратном порядке
- Ненайденные треки пропускаются
- Процесс отображается в консоли

# Внимание: Не публикуйте token!
